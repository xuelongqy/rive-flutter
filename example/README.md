# Rive Flutter Example

A demo application showcasing Rive Flutter.

## Getting Started

Clone this repository:
```bash
git clone https://github.com/rive-app/rive-flutter
```

Run the example:

```bash
cd rive-flutter/example
flutter pub get
flutter run
```
